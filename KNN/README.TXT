How to setup:
	import the .zip file as existing project to workspace
	build configure path
	import weka.jar file as referenced lib
	
How to get the results:
	When you run the Knn.java, it will output the results in console.
	Copy and parse the results from console. And write them into .dat files
	
*package p1: codes for doing J48 methods
*package p2: codes for implementing KNN and relief methods
*results folder: contains all the results for the project
.png files are results from gnuplots

For analysis of the project, check ReportforProject1.pdf in results folder.
